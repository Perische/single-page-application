# Wordly Dictionary SPA

## Project Description

Wordly Dictionary is a Single Page Application (SPA) built using HTML, CSS, and JavaScript. The application allows users to search for words and retrieve information such as definitions, pronunciation, audio, examples, and synonyms without refreshing the page.

---

## Features

✅ Search for words  
✅ Fetch data from an external dictionary API  
✅ Display word definitions  
✅ Display pronunciation  
✅ Play pronunciation audio  
✅ Show examples of word usage  
✅ Display synonyms  
✅ Handle invalid searches and API errors  
✅ Responsive user interface  

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- Free Dictionary API
